# TFM - Análisis de Clustering de Población

## Descripción
Aplicación web para analizar el crecimiento poblacional utilizando algoritmos de clustering como K-Means y DBSCAN.

## Funcionalidades
- Visualización de datos en mapa
- Aplicación de múltiples algoritmos
- Comparación de resultados
- Filtros dinámicos

## Tecnologías
- JavaScript
- HTML/CSS
- Algoritmos de clustering

## Cómo ejecutar
1. Abrir index.html en el navegador
